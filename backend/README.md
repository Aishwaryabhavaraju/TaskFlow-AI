# TaskFlow AI

AI-powered collaborative project management platform.

## Backend Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

## Features

- Authentication
- User Management
- Team Management
- Projects
- Boards
- Tasks
- Comments
- Notifications

## Installation

```bash
npm install
npm run dev
```

## Environment Variables

Create a `.env` file using `.env.example`.
